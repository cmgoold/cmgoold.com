Hi, I'm Conor. This is my small slice of the internet to write about statistics, software engineering, science, and a smattering of more informal topics too.

I live in North East England, working remotely as a data scientist. This site started in early 2025 as an informal New Year Resolution to practice writing and learn more web development. I was inspired by the authenticity of the IndieWeb movement, and from reading many great personal blogs on platforms such as Bear Blog.

All the code for this website be found at the Github repository. If you notice any errors or just want to reach out, please open an issue on Github.
